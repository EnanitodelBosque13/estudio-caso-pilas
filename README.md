# Estudio de caso: uso de pilas en expresiones aritméticas

Este proyecto corresponde al primer estudio de caso del curso Estructuras de Datos. El objetivo es implementar una pila en Java para apoyar el análisis de expresiones aritméticas.

La funcionalidad seleccionada fue el análisis de expresiones aritméticas, ya que las pilas permiten controlar elementos como paréntesis, operadores y operandos durante el proceso de validación.

## Funcionalidad del programa

El programa permite ingresar una expresión aritmética y determinar si su estructura es válida o inválida.

Ejemplos de expresiones válidas:

```text
(2 + 4) * (4 + 6)
contador + 5
a * (b + 3)
